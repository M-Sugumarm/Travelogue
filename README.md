## What I built
- Responsive React app (Vite)
- Search, favorites (localStorage), modal details
- Clean component structure and simple styles

## Tech stack
- React 18 (Vite)
- react-router-dom
- CSS for styling

## Setup
1. Install dependencies:

```powershell
npm install
```

2. Run dev server:

```powershell
npm run dev
```

3. Build for production:

```powershell
npm run build; npm run preview
```

## Notes & assumptions
- Data is local (src/data/trips.json). Replace with API if needed.
- Images use Unsplash links (hotlink) for demo only.
- No authentication implemented.
